## 📌 Project Overview

This project performs a complete data analysis pipeline on
Walmart's weekly sales dataset. It identifies key sales
drivers — including store type, department, and holiday
effects — and translates raw numbers into actionable
business insights.

## ✨ Key Features

- **Data Cleaning** — handles missing values, outliers,
  and data type inconsistencies
- **EDA** — sales trend analysis across 45 stores and
  99 departments
- **Visualization** — heatmaps, bar charts, and time
  series plots using Seaborn + Matplotlib
- **SQL Insights** — 10+ business queries for revenue
  ranking, holiday impact, and store performance

---

## 📊 Key Findings

- Holiday weeks drive **~7% higher** average weekly sales
- Store Type A consistently outperforms Types B and C
- Departments 92 and 95 are the highest revenue generators
