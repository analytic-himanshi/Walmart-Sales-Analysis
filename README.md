# Walmart Sales Analysis

This project performs an end-to-end data analysis on Walmart's weekly sales data.

## Project Overview
The objective of this project is to analyze the sales patterns and derive actionable business insights using Python and MySQL.

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Matplotlib, Seaborn
* **Database:** MySQL
* **Tools:** Jupyter Notebook

## Key Features
* Data Cleaning and Preprocessing to handle missing values and inconsistencies.
* Exploratory Data Analysis (EDA) to identify sales trends across different stores.
* Visualization of sales performance using heatmaps and bar charts.
* SQL queries to extract meaningful business insights from the dataset.

## Conclusion
The analysis highlights key factors impacting sales, such as store type and holiday impacts, providing a roadmap for better sales forecasting.
